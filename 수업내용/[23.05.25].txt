[#include <string.h>]

1. 길이구하기 : strlen() 
	- string length
	- char* 가 인자로 들어간다.
	- 인자값으로 들어온 문자열의 길이를 반환한다.
	
2. 문자열 비교 : strcmp()
	- string compare
	- 문자열 두개를 인자로 넣어준다.
	- 주의할점 : 같으면 0을 반환한다.
		일반적으로 '0'은 false 를 의미하는데 
		해당 함수는 같을때 '0'을 반환한다.
	- 틀리면 '-1'을 반환한다.
	
3. 값 복사 : strcpy()
	- string copy
	- 문자열 두개를 인자로 넣어준다.
	- 첫번째 인자가 destination(목적지)
	- 두번째 인자가 source(원천)
	- 오래된 함수다보니 안전하지 않다는 오류가 발생한다.
		때문에 'strcpy_s'로 사용해준다.
